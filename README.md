# darkest-space
Colorscheme darker for vim
## Preview



## Installation

To install this colorscheme, you have vundle or pathogen.
* Vundle:
    
    Plugin 'szorfein/darkest-space'

    And run :PluginInstall

* Pathogen:

    mkdir -p ~/.vim/bundle
    cd ~/.vim/bundle
    git clone https://github.com/szorfein/darkest-space
